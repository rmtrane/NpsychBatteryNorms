# Get Scaled Scores for use in T-score cacluations

Helper function to look up unadjusted scaled scores in the
[`ss`](https://rmtrane.github.io/NpsychBatteryNorms/reference/ss.md)
list for use in the fully adjusted T-scores.

## Usage

``` r
get_ss_score(raw_score, var_name)
```

## Arguments

- raw_score:

  numeric vector with raw scores

- var_name:

  string indicating what variable
