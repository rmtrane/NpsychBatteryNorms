# Calculate the Rey AVLT Accuracy

Calculates accuracy from true positives and false positives, but applies
[`valid_values_only()`](https://rmtrane.github.io/NpsychBatteryNorms/reference/valid_values_only.md)
first. As percentage, i.e. a value of 55 means 55%.

## Usage

``` r
calculate_reyarec(REYTCOR, REYFPOS)
```

## Arguments

- REYTCOR:

  true positives

- REYFPOS:

  false positives
